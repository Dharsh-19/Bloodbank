# Blood Bank Frontend Website

## 📌 Overview
A responsive Blood Bank web application designed to simplify the process of blood donation and access to blood availability. This project focuses on creating a clear, intuitive, and user-friendly interface to support donors and individuals in emergency situations.

## 🧩 Key Features
- User-friendly donor registration interface  
- Blood availability tracking UI  
- Emergency access flow for urgent needs  
- Fully responsive design for mobile and desktop users  

## 📄 Pages Included

### 🏠 Home Page
- Clean landing interface introducing the platform  
- Quick navigation to all major sections  
- Highlights key actions like Donate and Check Availability  

### 🩸 Blood Availability Page
- Displays available blood groups  
- Easy-to-read layout for quick access  
- Designed for fast decision-making in urgent situations  

### ❤️ Donate Page
- Donor registration form  
- Simple and minimal input fields for better usability  
- Focus on reducing friction in the donation process  

### 🚨 Emergency Page
- Quick access interface for urgent blood requirements  
- Prioritizes speed and clarity  
- Designed to help users act immediately
